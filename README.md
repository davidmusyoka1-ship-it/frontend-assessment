# Investment Club Frontend

This repository contains the frontend assessment project for the Investment Club.

## Features

- Responsive **membership card** with stats and profile.
- **Foundation** and **Economy** memberships toggle info on click.
- **Floating WhatsApp button** for instant contact.
- Footer with **All Rights Reserved**.
- Modal contact form.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
